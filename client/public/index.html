TopContrib

TopContrib is a simple web application that can be used to fetch the details of the most popular GitHub repositories of organizations and the top contributors to those repositories

Usage
  • Enter the name of the organization, limit to the number of popular repositories (n) and limit to the number of contributors (m)
  • Click on Go to fetch the top repositories and contributors
  • If you get an error saying GitHub has throttled requests because of rate limits, generate an access token in GitHub without any scope(s) and use that token to fetch the results
